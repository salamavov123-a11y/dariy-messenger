# Dariy Messenger (.exe)

Это desktop-версия приложения на Electron.

## Запуск локально

```bash
npm install
npm start
```

## Сборка Windows `.exe`

```bash
npm install
npm run build:win
```

После сборки появится папка:

- `dist/DariyMessenger-win32-x64/`

Файл приложения:

- `dist/DariyMessenger-win32-x64/DariyMessenger.exe`
